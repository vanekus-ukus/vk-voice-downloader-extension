# Дисклеймер

Этот проект является независимым учебным и практическим инструментом.

## Неаффилированность

Репозиторий не связан с VK, не одобрен VK и не поддерживается VK.

## Предполагаемые границы использования

Расширение предназначено для улучшения пользовательского опыта при работе с контентом, который уже виден пользователю в веб-интерфейсе VK.

Оно **не**:

- обходит авторизацию
- автоматизирует логин
- обращается к приватному API напрямую
- пытается обходить DRM или защитные механизмы платформы

## Замечание о стабильности

Проект опирается на DOM-структуру VK и React-данные страницы, которые могут измениться без предупреждения. Поломки из-за upstream-изменений интерфейса возможны и ожидаемы.

Пользователь сам отвечает за использование расширения в соответствии с правилами и условиями сервисов, которыми он пользуется.
