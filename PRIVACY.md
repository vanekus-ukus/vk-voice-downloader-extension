# Конфиденциальность

`VK Voice & Clips Downloader` работает локально в браузере пользователя.

## Что хранит расширение

- одно локальное значение, которое показывает, включено расширение или выключено

Эта настройка сохраняется через `chrome.storage.local`.

## Что расширение не собирает

- учётные данные
- сообщения
- список контактов
- аналитику
- телеметрию
- удалённую статистику использования

## Сетевое поведение

Само расширение не отправляет пользовательские данные на внешние серверы.

Единственное сетевое действие, которое возникает по инициативе расширения, — это браузерная загрузка файла, которую сам пользователь запускает со страницы VK, уже доступной ему в интерфейсе.

## Границы работы

Расширение работает только с тем контентом, который уже доступен пользователю в текущем отображаемом интерфейсе VK.
