# Участие в проекте

Спасибо за интерес к проекту.

## Локальная разработка

1. Склонируйте репозиторий.
2. Откройте `chrome://extensions` или `browser://extensions`.
3. Включите **Режим разработчика**.
4. Загрузите корень репозитория как распакованное расширение.
5. После изменений в файлах перезагрузите расширение.

## Принципы проекта

- Не добавлять сборщик и тяжёлую инфраструктуру без действительно веской причины.
- Предпочитать ясность и поддерживаемость красивым, но спорным абстракциям.
- Считать VK DOM и React internals нестабильными внешними зависимостями и писать защитный код.
- Не добавлять работу с приватным API, автоматизацию логина или обход авторизации.

## Ожидания к pull request

- Изменения должны быть узкими по объёму и практически обоснованными.
- При изменении поведения или структуры нужно обновлять документацию.
- Совместимость с Manifest V3 должна сохраняться.
- Не стоит усложнять проект тяжёлой инфраструктурой ради мелких улучшений.

## Особенно полезные направления

- повышение устойчивости извлечения для новых вариантов VK-разметки
- улучшение формирования имён файлов
- улучшение popup UX без лишнего усложнения
- улучшение документации, скриншотов и демо-материалов
