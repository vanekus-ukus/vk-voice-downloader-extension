# История изменений

Все заметные изменения проекта фиксируются в этом файле.

Формат вдохновлён [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
а версионирование следует [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-04-11

### Добавлено

- публичная структура репозитория для MV3-расширения
- инъекция кнопки скачивания рядом с голосовыми сообщениями VK
- извлечение URL через React Fiber / React props с fallback-путями
- поток скачивания через `chrome.downloads`
- popup с сохранением состояния ВКЛ/ВЫКЛ
- документация, privacy/disclaimer и метаданные для публичного GitHub-репозитория
